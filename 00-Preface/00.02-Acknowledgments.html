Acknowledgments

Acknowledgments#

The creation of this book and supporting material would not be possible without the gracious support of my wife and family. Students at Portland Community College continue to give me hope that the next generation of engineers will be a diverse group of team problem solvers.

The Python Data Science Handbook and Machine Learning in Python as well as Reiman Equations in Python served as inspiration and examples of using Jupyter notebooks to construct a book. The bookbook repository on GitHub provided a starting point for the tooling used to convert this book from Jupyter notebooks into LaTeX for printing. The Executable Books project and the Jupyter Book package was used to convert the source Jupyter notebooks into the book website for the 3.9 Edition.
